# AI 应用开发就业方向

> 面向有 Claude / Agent 开发经验的工程师 | 当前位置：西安，中国

---

## 一、核心就业方向

### 1. AI 应用工程师（大模型应用开发）

**做什么：** 用国内外 LLM API 构建产品功能，对接业务需求

- 设计和实现 AI 功能：智能搜索、对话系统、自动化流程
- Prompt 设计与优化
- RAG 系统搭建（结合企业私有知识库）
- 与产品/业务团队协作

**市场需求：** 极高，几乎所有互联网公司都在招
**薪资参考（西安）：** 20k–35k/月（有大厂/AI创业公司经验可到 40k+）
**薪资参考（远程/北上深广）：** 30k–60k/月
**典型雇主：** 字节跳动西安研发中心、三星西安、各 AI 创业公司

---

### 2. AI Agent / 自动化工程师

**做什么：** 构建自主 Agent，替代人工完成复杂任务流程

- 设计多步骤 Agent 工作流
- 工具调用（Function Calling）/ MCP Server 开发
- 业务流程自动化（审批、报表、客服、代码审查等）
- 对接企业内部系统（ERP、CRM、OA）

**市场需求：** 快速增长，2025–2026 是国内爆发期
**薪资参考（西安）：** 20k–35k/月
**你的优势：** 已有 Agent Skill 开发经验，在国内这个方向人才极度稀缺

---

### 3. AI 平台工程师

**做什么：** 构建供内部工程师使用的 AI 基础设施

- 企业 AI 网关（统一模型调用入口、限流、计费、日志）
- 内部 Prompt 管理平台、Eval 评估系统
- Agent 运行时框架
- 模型接入适配（国内：文心、通义、智谱；国外：Claude、GPT）

**市场需求：** 高，中大型公司都在建内部 AI 中台
**薪资参考（西安）：** 25k–45k/月
**技能要求：** 扎实后端工程能力 + AI 产品理解

---

### 4. AI 产品经理（技术型 PM）

**做什么：** 定义 AI 产品方向，连接技术与业务

- 评估大模型能力边界，制定产品路线图
- 设计 AI 功能的用户体验
- 定义评估指标，推动 Prompt/模型迭代
- 管理 AI 风险（幻觉、合规、数据安全）

**市场需求：** 高，有技术背景的 AI PM 在国内非常稀缺
**薪资参考（西安）：** 20k–40k/月
**你的优势：** 实际开发经验让你比传统 PM 更懂技术边界，转型门槛低

---

### 5. AI 解决方案架构师 / 售前

**做什么：** 帮助企业客户制定和落地 AI 方案

- 评估客户业务，识别 AI 落地机会
- 设计技术方案（选型、架构、成本估算）
- POC 交付，推动采购决策
- 典型雇主：华为云、阿里云、腾讯云、百度智能云、科大讯飞

**市场需求：** 高，企业 AI 转型需求旺盛
**薪资参考（西安）：** 20k–35k/月 + 提成/奖金

## 二、西安本地机会分析

### 西安本地主要雇主

| 公司/类型 | 方向 | 备注 |
|----------|------|------|
| 字节跳动西安 | 后端/AI 工程 | 规模大，薪资对标北京 |
| 三星西安研发中心 | AI 算法/应用 | 外企待遇，相对稳定 |
| 科大讯飞西北分支 | 语音/NLP 应用 | 本土 AI 龙头 |
| 陕西本地 AI 创业公司 | 各类应用开发 | 数量增长快，期权机会 |
| 政府/高校数字化项目 | AI 应用集成 | 合同制，稳定但薪资一般 |
| 华为/中兴西安研究所 | 平台/基础设施 | 大厂福利，节奏较快 |

### 西安的现实情况

**优势：**
- 生活成本低，同等薪资实际购买力更强
- 西安是西北最大科技中心，近年来 AI 岗位增长明显
- 西北大学、西安交大等高校资源，技术氛围好
- 部分大厂西安研发中心薪资对标总部

**劣势：**
- AI 顶级岗位仍集中在北上深杭，西安选择相对少
- 本地 AI 公司规模较小，技术深度有限
- 高端 AI 岗位竞争者中有大量名校应届生

## 三、远程工作机会（强烈推荐关注）

AI 应用开发是远程友好度最高的岗位之一。身在西安可以做北京/上海/深圳薪资的工作：

- **全远程岗位**：国内 AI 创业公司、海外华人团队
- **混合办公**：大厂西安研发中心（字节、华为等）
- **外包/项目制**：给企业做 AI 转型项目，接单灵活

**找远程岗位的渠道：**
- 拉勾网（筛选"远程"标签）
- Boss 直聘（"大模型"+"可远程"）
- 稀土掘金招聘板块
- V2EX 的求职版块（技术社区，质量高）
- 即刻 App（AI 圈子活跃）

## 四、国内技术栈注意事项

国内 AI 应用开发与海外有一些差异：

| 方面 | 国内常用 | 备注 |
|------|---------|------|
| 基础模型 | 通义千问、文心一言、智谱 GLM、DeepSeek | DeepSeek 2025年后影响力大增 |
| 向量数据库 | Milvus（国产）、Chroma、pgvector | Milvus 是国内事实标准 |
| 开发框架 | LangChain、LlamaIndex、Dify（国产）| Dify 在国内中小企业很流行 |
| 部署环境 | 阿里云、腾讯云、华为云 | 需要了解国内云的 AI 服务 |
| 合规要求 | 数据不出境、大模型备案 | 企业采购时的硬性要求 |

> Claude 的使用经验在国内也是竞争优势，但要同时掌握国内主流模型 API

## 六、求职实用建议

### 简历突出点
- 量化成果：用 Agent 节省了多少人工时间、处理了多少文档
- 展示 Tool Use / MCP / RAG 经验，国内这些关键词含金量极高
- 提及熟悉国内主流模型（即使是通过 API 对比测试）
- GitHub 有可展示的 AI 项目

### 面试准备
- **系统设计**：设计一个企业知识库 RAG 系统 / 多 Agent 自动化流程
- **Prompt 调试**：给你一个效果差的 Prompt，如何优化
- **工程问题**：如何评估 AI 输出质量，如何处理幻觉
- **国内合规**：大模型备案是什么，数据安全注意事项

### 西安求职渠道
- Boss 直聘（最推荐，AI 岗位多）
- 拉勾网（技术岗位聚集）
- 猎聘（中高级岗位）
- 西安本地技术社群（微信群、掘金西安站）
